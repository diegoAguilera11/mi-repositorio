Fashion Dog

Sistema web para una perruqueria de la región de Antofagasta, Chile.
Los clientes pueden solicitar servicios a domicilio y dejar un respectivo comentario de satisfacción del servicio una vez haya finalizado. Tambien los estilistas pueden realizar servicios a domicilio.
Por ultimo el sistema cuenta con una área de gestión dedicada al administrador de Fashion Dog.

Caracteristicas:

  • Tipo de Aplicación: Web
  • Framework: Laravel
  • Base de datos: Sql
  • Principal Lenguaje de programación: PHP

Restaurant APP

Sistema móvil para un restaurant de la region de Antofagasta, Chile.
Los clientes deben seleccionar la mesa en la cual estan ubicados para luego realizar su orden con los productos disponibles. Tambien desde la aplicación el administrador puede gestionar los productos y categorias del restaurante. Por ultimo el sistema cuenta con un apartado web para gestionar los pedidos ingresados.

Características:

  • Tipo de Aplicación: Móvil
  • Libreria: React Native
  • Rest Api: Laravel
  • Base de datos: Sql
  • Principal Lenguaje de Programación: JavaScript y PHP
  • Fecha de inicio: 10/04/2022
  • Fecha de termino: 05/12/2022
